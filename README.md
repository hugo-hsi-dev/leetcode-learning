# LeetCode Interview Prep — From Zero CS to Medium-Level Problems

A self-study curriculum for JavaScript/TypeScript developers with no computer science background, targeting the ability to solve LeetCode Easy and Medium problems for technical interviews.

## Who This Is For

- You're comfortable with JS/TS, React, and Node.js
- You have no formal CS education (no algorithms class, no data structures course)
- You want to pass technical interviews at companies that ask LeetCode-style questions
- You struggle with Easy problems and want to build up to solving Mediums consistently

## How to Use This Curriculum

1. **Go in order.** The curriculum is sequenced so each topic builds on previous ones. Don't skip ahead.
2. **Read the lesson first**, then do the exercises. Don't look at solutions until you've genuinely tried.
3. **Code examples are in both TypeScript and Python.** TS is your primary language; Python is included so you get comfortable reading it (you'll see it constantly in LeetCode discussions and editorials).
4. **Do the warm-up exercises before the LeetCode problems.** The warm-ups isolate one concept. LeetCode problems combine multiple concepts.
5. **Use an AI assistant to teach from these files.** Each lesson is written as structured notes for an AI tutor — point it at any lesson file and say "teach me this" or "quiz me on this." The outlines give the AI enough structure to deliver a full lesson while you ask questions and go deeper.

## Conventions Used

- **TypeScript** is the primary language for all code examples
- **Python** is shown alongside every TypeScript example for reading familiarity
- Each lesson starts with **Prerequisites** and is structured as detailed outlines for AI-assisted learning
- Practice problems are tagged: `[Warm-up]`, `[Easy]`, `[Medium]`
- LeetCode problems include problem number, title, link, and a hint
- Solution files include detailed comments explaining the *why*, not just the *what*
- Every solution includes Big O time and space analysis

## After This Curriculum

Once you can solve Easy problems consistently and Medium problems with some effort:

1. Work through the [Blind 75](https://leetcode.com/discuss/general-discussion/460599/blind-75-leetcode-questions) problem list
2. Move to [NeetCode 150](https://neetcode.io/practice) for more structured practice
3. Start timed practice sessions (25-30 minutes per problem)
4. Practice explaining your solutions out loud (mock interview style)
